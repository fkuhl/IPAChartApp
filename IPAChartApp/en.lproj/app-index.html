    • Install the keyboard

    Like other iOS third-party keyboards, IPAChartBoard comes to your device in a carrying app, which you've just installed. Now install IPAChartBoard through System Settings to make it available to other apps:
    1. Settings
    2. General
    3. Keyboard
    4. Keyboards
    5. Add New Keyboard
    6. IPAChartBoard
    7. Tap "IPAChartBoard - IPAChartApp"

    From your default system keyboard, you switch to IPAChartBoard by tapping the key, as you would to use any other system keyboard.
